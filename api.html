API

USE THE VAULT FROM YOUR OTHER PROJECTS

OVERVIEW

The Vault is a static site, but it's fully data-driven. Every clip's path is predictable, every clip's metadata is in manifest.json, and the vault page itself accepts URL parameters for deep-linking from external apps. This page documents the contract.

NOTE: Once deployed to Netlify, replace the local paths below with your live URL (e.g., https://duke-nukem-vault.netlify.app). Netlify is configured with permissive CORS headers on /clips/* and /manifest.json, so you can hotlink directly from any other web project.

MANIFEST SCHEMA

The single source of truth is manifest.json. It contains everything a client needs.

{ "version": 1, "name": "Duke Nukem Audio Vault", "total": 226, "categories": ["catchphrase", "combat", "damage", ...], "sources": ["Atomic Soundboard", "Ultimate Soundboard", ...], "clips": [ { "id": "clip_001", // stable identifier "slug": "come-get-some", // kebab-case URL-safe "title": "Come get some", // human-readable "transcript": "Come get some", "category": "catchphrase", // 11 possible values "source": "Atomic Soundboard", "source_key": "soundboard_atomic", "path": "clips/catchphrase/come-get-some.mp3", "filename": "come-get-some.mp3", "type": "audio", "size": 23436, // bytes "format": "mp3" }, ... ] }

FETCH AND USE

JavaScript — vanilla browser

const VAULT = 'https://your-vault.netlify.app'; const data = await fetch(`${VAULT}/manifest.json`).then(r => r.json()); // Random catchphrase on a button click const catchphrases = data.clips.filter(c => c.category === 'catchphrase'); button.addEventListener('click', () => { const clip = catchphrases[Math.floor(Math.random() * catchphrases.length)]; new Audio(`${VAULT}/${clip.path}`).play(); });

React component

function DukeButton({ category = 'catchphrase' }) { const [clips, setClips] = useState([]); useEffect(() => { fetch('https://your-vault.netlify.app/manifest.json') .then(r => r.json()) .then(d => setClips(d.clips.filter(c => c.category === category))); }, [category]); const playRandom = () => { const clip = clips[Math.floor(Math.random() * clips.length)]; new Audio(`https://your-vault.netlify.app/${clip.path}`).play(); }; return <button onClick={playRandom}>Hell Yeah</button>; }

Direct embed — no JavaScript

<audio src="https://your-vault.netlify.app/clips/catchphrase/come-get-some.mp3" controls></audio>

DEEP LINK URL PARAMETERS

The library page (vault.html) accepts these query parameters. Useful for opening the vault pre-filtered or auto-playing a specific clip from another app.

ParameterEffectExample
?clip=clip_001 Open vault, auto-play that clip vault.html?clip=clip_005
?category=combat Filter to a specific category vault.html?category=catchphrase
?source=Atomic+Soundboard Filter to a specific source vault.html?source=Myinstants
?search=come+get Pre-fill the search box vault.html?search=hail+to+the+king

Parameters can be combined: vault.html?category=catchphrase&search=king

FILE PATHS

All audio lives under /clips/<category>/<slug>.mp3. Predictable. URL-safe. Cached at the edge for one year on Netlify (Cache-Control: public, max-age=31536000, immutable).

RATE LIMITS

None. It's a static site on Netlify's CDN. Hammer it.

VOICE FORGE — PROGRAMMATIC USE

The Voice Forge logic is contained in forge.html. To replicate the word-stitching algorithm in another app:

// Build a word index from the manifest const wordIndex = new Map(); manifest.clips.forEach(clip => { const words = (clip.transcript || clip.title || '') .toLowerCase().match(/\b\w+\b/g) || []; words.forEach(w => { if (!wordIndex.has(w)) wordIndex.set(w, []); wordIndex.get(w).push({ clip, length: words.length }); }); }); // Find best clip for a word (shortest transcript = most isolated) function findClipForWord(word) { const matches = wordIndex.get(word.toLowerCase()); if (!matches) return null; matches.sort((a, b) => a.length - b.length); return matches[0].clip; }

ELEVENLABS SETUP — REAL DUKE AI VOICE

The Voice Forge has an AI VOICE mode that uses ElevenLabs for true text-to-speech. To turn it on, set two environment variables in Netlify and you're done.

Step 1 — Get your ElevenLabs API key

Sign in at elevenlabs.io → click your profile in the top right → Profile + API Key. Copy the key (starts with sk_...).

Step 2 — (Optional but recommended) Clone Duke's voice

In ElevenLabs:

  1. Go to Voices → Add a new voice → Instant Voice Clone
  2. Name it Duke Nukem
  3. Upload 5–10 of the cleanest catchphrase clips from this vault. Good picks: come-get-some.mp3, hail-to-the-king.mp3, balls-of-steel.mp3, damn-im-good.mp3, kick-ass-and-chew-bubblegum.mp3
  4. Submit. ElevenLabs trains the voice in 30–60 seconds.
  5. Once created, click the voice and copy its Voice ID (looks like 21m00Tcm4TlvDq8ikWAM)

Step 3 — Set the env vars in Netlify

Go to Netlify → Site settings → Environment variables and add:

ELEVENLABS_API_KEY = sk_your_actual_key_here ELEVENLABS_VOICE_ID = your_duke_voice_id // optional default

Click Save. Netlify will redeploy automatically (~30 seconds). The Voice Forge AI mode will pick it up.

Step 4 — Use it

Open Voice Forge → click the 🧠 AI VOICE (ELEVENLABS) tab. The status bar should turn gold and show "Connected." Pick your Duke voice from the dropdown, type any text, hit GENERATE.

Endpoints

POST /.netlify/functions/duke-tts { text, voiceId, settings? } → returns audio/mpeg

Use from another project

const response = await fetch('https://duke-nukem-vault.netlify.app/.netlify/functions/duke-tts', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ text: 'Damn, I am good.', voiceId: 'YOUR_DUKE_VOICE_ID' }) }); const audioBlob = await response.blob(); const url = URL.createObjectURL(audioBlob); new Audio(url).play();
NOTE ON RATE LIMITS: ElevenLabs counts characters against your monthly quota. The free tier is 10,000 chars/month, Starter is 30,000, Creator is 100,000. Each GENERATE click in the Forge spends characters equal to the length of your text.
