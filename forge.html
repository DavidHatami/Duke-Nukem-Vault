HONEST ABOUT WHAT THIS IS: This is not AI voice cloning. It uses Duke's actual recorded clips as building blocks. Type a phrase — the forge searches the corpus for matches, breaks unmatched text into words, and stitches clips that contain those words. Words not in the library fall back to browser speech synthesis (which is NOT Duke). For exact phrases that exist in the corpus, output is real Duke. For arbitrary text, output quality depends on what's in the library.
